# Vel Tech Summer Internship

## Day 1 - Python Basics

This repository contains the tasks and programs completed during my Summer Internship at Vel Tech University.

### Programs Implemented

#### 1. BMI Calculator
A Python program that:
- Takes weight (kg) and height (cm) as input.
- Calculates Body Mass Index (BMI).
- Categorizes the user as:
  - Underweight
  - Normal Weight
  - Overweight

**Formula Used:**
BMI = Weight (kg) / Height² (m²)

#### 2. Birth Year and Future Year Calculator
A Python program that:
- Takes the user's birth year as input.
- Calculates the current age based on the year 2026.
- Predicts the year after 10 years.

#### 3. Student Information Display
A simple Python program using variables and formatted strings to display:
- Name
- Age
- College
- Branch
- CGPA
