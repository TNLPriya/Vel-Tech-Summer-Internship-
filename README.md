## Day 1 - Python Basics

### Objective
To learn the fundamentals of Python programming, including variables, user input, arithmetic operations, conditional statements, and formatted output.

### Programs Implemented

### 1. BMI Calculator
A Python program that:
- Accepts weight (kg) and height (cm) as input.
- Calculates the Body Mass Index (BMI).
- Classifies the result as:
  - Underweight
  - Normal Weight
  - Overweight

### 2. Birth Year and Future Year Calculator
A Python program that:
- Accepts the birth year as input.
- Calculates the current age based on the year 2026.
- Predicts the age after 10 years.

### 3. Student Information Display
A simple program using variables and f-strings to display:
- Name
- Age
- College
- Branch
- CGPA
