**Day 11 - Student Registration System**
Project Overview
This project is a Student Registration System developed using Flask, HTML, and CSS. The application includes multiple pages connected using Flask routing and render_template().
**Features**
Home Page
Student Registration Page
Students List Page
About Page
Flask Routing
Navigation Bar
Responsive Design
Form Styling
Table Styling
Dummy Student Data Display
**Project Structure**
