Day 10 - Spam Detection Project

Project Overview

This project is a machine learning-based Spam Detection System developed as part of the Vel Tech Summer Internship (Track 2). The model classifies messages as Spam or Not Spam using text classification techniques.

Dataset Used

Spam message dataset containing text messages labeled as spam or not spam.

Technologies Used

- Python
- Google Colab
- Scikit-learn
- Pandas
- NumPy
- Pickle

Machine Learning Model

Algorithm Used: Random Forest Classifier

Project Workflow

1. Data Loading
2. Data Preprocessing
3. TF-IDF Vectorization
4. Model Training
5. Model Evaluation
6. Custom Message Testing
7. Saving Model as "final_model.pkl"

Files Included

- "Spam_Detection_Project.ipynb" – Project notebook
- "dataset.csv" – Dataset used for training
- "final_model.pkl" – Saved trained model for Flask integration

Output

The model predicts whether a message is Spam or Not Spam based on input text.

Future Scope

This model is prepared for Flask integration in Track 3 for deployment as a web application.
